## Pobby

[pobby]https://github.com/user-attachments/assets/e86a7716-785e-4523-816a-4ac394359375

Pobby is a CLI tool that displays and manage listening ports. 

## Installation
```bash
go install github.com/bettaburger/pobby/cmd/pobby@latest
```
Check installation
```bash
ls -l ~/go/bin/pobby
```
## Executable Set up
```bash
sudo mv ~/go/bin/pobby /usr/local/bin
```
Set permission
```bash
sudo chmod +x /usr/local/bin/pobby
```

## Starting up the tool
Simply run the program with this command
```bash
pobby list
```
