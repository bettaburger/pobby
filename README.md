## Pobby

![pobby](https://media3.giphy.com/media/v1.Y2lkPTc5MGI3NjExMTRhNmt6YWM5YzdrdmgwZHdsOWRmM3J4MWx0cG52ZWI0cXR6NnAyYyZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/SjYmS3k7j3mDYJg1tm/giphy.gif)

Pobby is a CLI tool that displays and manage listening ports. 

## Installation
```bash
go install github.com/bettaburger/pobby/cmd/pobby@latest
```
Check installation
```bash
ls -l ~/go/bin/pobby
```
## Executable Set up
```bash
sudo mv ~/go/bin/pobby /usr/local/bin
```
Set permission
```bash
sudo chmod +x /usr/local/bin/pobby
```

## Starting up the tool
Simply run the program with this command
```bash
pobby list
```
